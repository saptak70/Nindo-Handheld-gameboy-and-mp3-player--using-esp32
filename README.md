# RetroHand — ESP32-S3 Retro Gaming + BT Audio Handheld

A custom handheld device built from scratch — PCB, firmware, 
and enclosure. Plays NES/DOOM, streams audio to BT headphones.

## Features
- ESP32-S3-WROOM-1 N8R8 (dual core, BT5, 8MB PSRAM)
- ST7789V3 1.69" 240×280 SPI display
- PCM5102A I2S DAC + 3.5mm jack + BT A2DP
- MicroSD for ROMs and music
- LiPo battery with TP4056 charging + DW01A protection
- Custom KiCad PCB + 3D printed enclosure
